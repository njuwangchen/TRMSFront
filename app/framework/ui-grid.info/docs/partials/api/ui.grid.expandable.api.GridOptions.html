GridOptions (api in module ui.grid.expandable )

Description

Options for configuring the expandable feature, these are available to be
set using the ui-grid gridOptions

Properties

  • Width in pixels of the expandable column. Defaults to 40

    Example

    $scope.gridOptions = {
  expandableRowHeaderWidth: 40
}

  • enableExpandable

    Whether or not to use expandable feature, allows you to turn off expandable on specific grids within your application, or in specific modes on this grid. Defaults to true.

    Example

    $scope.gridOptions = {
  enableExpandable: false
}

  • enableExpandableRowHeader

    Show a rowHeader to provide the expandable buttons. If set to false then implies you're going to use a custom method for expanding and collapsing the subgrids. Defaults to true.

    Example

    $scope.gridOptions = {
  enableExpandableRowHeader: false
}

  • expandableRowHeight

    Height in pixels of the expanded subgrid. Defaults to 150

    Example

    $scope.gridOptions = {
  expandableRowHeight: 150
}

  • expandableRowTemplate

    Mandatory. The template for your expanded row

    Example

    $scope.gridOptions = {
  expandableRowTemplate: 'expandableRowTemplate.html'
}
